[h1]Traduzione italiana di Quasimorph[/h1]


Aggiunge una traduzione in italiano al gioco.
Attualmente la traduzione è automatica (effettuata da una macchina) e funziona solo con la versione beta instabile.

[h1]Aiuta con la traduzione[/h1]

Al momento il testo è tradotto automaticamente.

Puoi contribuire a correggere la traduzione tramite GitHub all'indirizzo https://github.com/NBKRedSpy/QM-ItalianTranslation. Il file di localizzazione è [i]./src/localization.tsv[/i]. Se preferisci, puoi anche contattarmi sul server Discord ufficiale del gioco.

Per testare le modifiche in locale, modifica il file situato in <SteamDir>\steamapps\workshop\content\2059170\3758584967\localization.tsv. Sostituisci <SteamDir> con la cartella di installazione di Steam.

Effettua prima una copia di backup del file, poiché Steam potrebbe sovrascriverlo quando la mod viene aggiornata.

[h2]Esempio di file di localizzazione[/h2]

Per facilitare la traduzione, il file denominato [i]%UserProfile%\AppData\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\ItalianTranslation\localization_export.tsv[/i] conterrà una versione inglese del file di localizzazione. Questo file verrà sovrascritto se il gioco apporterà modifiche alla localizzazione.

[h1]Configurazione[/h1]

Il file di configurazione verrà creato al primo avvio del gioco e si trova in [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\ItalianTranslation\config.json[/i].
[table]
[tr]
