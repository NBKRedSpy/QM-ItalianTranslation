[h1]Quasimorph ItalianTranslation[/h1]


An Italian Translation mod.
Note that this is currently machine translated.

[h1]Translation Export[/h1]

A copy of the game's translation text will be written to
[i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\ItalianTranslation\localization_export.tsv[/i] for reference.

[h1]Configuration[/h1]

The configuration file will be created on the first game run and can be found at [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\ItalianTranslation\config.json[/i].
[table]
[tr]
[td]Name
[/td]
[td]Default
[/td]
[td]Description
[/td]
[/tr]
[tr]
[td]ExportData
[/td]
[td]true
[/td]
[td]Exports the game's localization text to the configuration directory.
[/td]
[/tr]
[/table]
